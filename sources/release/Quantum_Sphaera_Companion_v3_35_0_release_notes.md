## Release type

Public release-final package built from the v3.35.0-r04 release candidate.

## Main changes

- Keeps the Active Core / Historical Witness split-bundle architecture.
- Adds pre-v4 QAM Hilbert-shift planning while reserving the actual global QAM tag migration for v4.0.0.
- Defines guarded mixed QAM readouts as derived packages, not primitive mixed tensors.
- Adds an optional binary / IPv4-like mask registry for readout classification.
- Introduces a conservative two-sided structural tensor notation for QAM/MML/MMA readout grammar.
- Stabilizes the notation through role-erasure, guarded mixed-readout normal form, mixed-readout expansion, Boolean selector grammar, mask-filter conservativity, and proof-neutrality.
- Adds DOI-bearing release-final metadata and a v3.35.0 Version/DOI record.

## Non-claims

- No global QAM Hilbert-shift / reindexing is performed.
- No primitive mixed tensors are activated.
- No OP 1/4 or OP 5(B) closure is claimed.
- No Turing-completeness claim is added.
- The two-sided notation remains a conservative compression grammar, not a new tensor calculus.
