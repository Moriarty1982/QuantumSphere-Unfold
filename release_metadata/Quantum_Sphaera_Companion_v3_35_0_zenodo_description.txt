This release is a pre-v4 QAM/tensor preparation release of the Quantum Sphaera Companion. It keeps the split-bundle architecture introduced in v3.34.0: the Active Core is the governing current proof line, while the Historical Witness is preserved as a separate proof-memory and genealogy PDF.

The main changes are conservative and preparatory. The release adds the short/extended abstract architecture, documents the Hilbert-hotel style QAM reindexing plan reserved for a later v4 major-version migration, defines guarded mixed QAM readouts as derived packages rather than primitive mixed tensors, and records an optional binary/IPv4-like mask registry for readout classification. It also introduces and stabilizes a two-sided structural tensor notation for local QAM/MML/MMA readout grammar: front-side indices mark readout/frame/language interfaces, back-side indices mark structure/action/field interfaces, ordinary upper-lower contraction remains classical action/readout, and same-variance pairings are permitted only through explicit comparison or selector tensors.

The r03/r04 layers stabilize this notation by adding role-erasure back to ordinary QAM-over-ZF role data, guarded mixed-readout normal form, mixed-readout expansion, Boolean selector grammar, mask-filter conservativity, LLM/audit role tables, and a release-candidate freeze/audit layer. The present v3.35.0 release-final pass inserts the assigned version-specific DOI and packages the release files, audit report, release notes, manifest, checksums, and diffs.

No global QAM Hilbert-shift is performed in this version. No primitive mixed tensors are activated. No OP 1/4 or OP 5(B) closure is claimed. The two-sided notation is a conservative compression grammar and LLM-stability layer, not a replacement for classical tensor calculus and not a new closure criterion.
