# v3.31.0 full monolith source snapshot

This directory contains a reconstructable GitHub source snapshot of the full v3.31.0 monolith.

## Why chunked

The monolith is stored in chunked TeX files so the full 700+ page source can live in the repository in a manageable, inspectable form.

## Reconstruction

Use `Quantum_Sphaera_Companion_v3_31_0_full_monolith_repo_wrapper.tex` as the assembly wrapper. It inputs the chunk files in order and reconstructs the full monolithic source snapshot.

## Status

This is a repository source snapshot pushed after the public v3.31.0 release. It is intended for inspection, tagged recovery, and source preservation in the GitHub mirror.
